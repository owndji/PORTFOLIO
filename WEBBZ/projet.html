HANGMAN

Le projet d'école consistait à développer un jeu de Hangman en utilisant le langage de programmation Go, avec une interface basée sur le terminal. L'objectif était de créer une expérience de jeu interactive entièrement réalisée en ligne de commande, mettant en avant les fonctionnalités du jeu Hangman tout en démontrant les capacités de Go dans le développement d'applications de terminal. Ce projet permettait ainsi aux étudiants d'explorer les concepts de programmation, la manipulation de chaînes de caractères et les interactions utilisateur en utilisant Go dans un environnement de développement simple et accessible.

HANGMAN-WEB

Le projet d'école visait à créer un jeu de Hangman en utilisant HTML et CSS pour la partie frontale du site, tandis que le back-end était développé en Go, ajoutant ainsi une dimension interactive et dynamique à l'expérience de jeu. L'objectif était de combiner les langages de programmation web avec Go pour offrir une expérience utilisateur immersive et divertissante, tout en renforçant les compétences en développement web et en explorant les capacités polyvalentes de Go dans le domaine du développement d'applications.

Groupie-Tracker

Le projet Groupie Tracker avait pour objectif de concevoir une application similaire à Spotify en utilisant une API spécifique, avec le développement réalisé sur le logiciel Fyne en GoLang. L'application visait à offrir une expérience utilisateur immersive, permettant aux utilisateurs d'accéder à une vaste bibliothèque musicale, de découvrir de nouveaux artistes et de créer des listes de lecture personnalisées. En utilisant Fyne, les développeurs ont pu concevoir une interface graphique élégante et conviviale, tout en exploitant les fonctionnalités de l'API pour fournir un contenu musical dynamique et actualisé. Ce projet a permis aux étudiants d'explorer les capacités de Go dans le développement d'applications graphiques tout en se familiarisant avec l'intégration d'API externes pour enrichir les fonctionnalités de leur application.
